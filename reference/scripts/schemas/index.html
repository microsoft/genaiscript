Data Schemas

It is possible to force the LLM to generate data that conforms to a specific schema. This technique works reasonably well and GenAIScript also provides automatic validation â€œjust in caseâ€.

You will notice that the schema supported by GenAIScript is much simpler than the full-blow JSON schema specification. We recommend using simple schemas to avoid confusing the LLM; then port them to your application specific data format later on.

defSchema

Use defSchema to define a JSON/YAML schema for the prompt output.

You can change this behavior by using the { format: "json" } option.

The trace allows you to see the schema source and the rendered prompt and the cli will write the generated TypeScript files in the output folder as well.

Validation
